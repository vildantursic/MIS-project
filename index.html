Welcome to project MIS.

Bellow simple stems how to get project and run it:

$ git clone https://github.com/vildantursic/MIS-project.git
$ nmp install -g serve

Project will be available on your local server port 3000, localhost:3000

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
